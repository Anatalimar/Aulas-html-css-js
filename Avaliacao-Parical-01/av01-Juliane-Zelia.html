hypertexto Markup language

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a publicação na Web. Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendência em tomar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), OS navegadores ainda hoje conseguem interpretar páginas web que estão longe de ser um código HTML válido.
