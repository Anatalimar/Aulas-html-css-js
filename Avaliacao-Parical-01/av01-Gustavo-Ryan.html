HyperText Markup Language


As primeiras versões do HTML foram com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com apublicação na Web.Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas(retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web que estão longe de ser um código HTML válido.
