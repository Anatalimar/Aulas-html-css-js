HyperText Markup Language



As primeiras versões do HTML foram definidas com regras sintaticas flexíveis, o que ajudou aqueles sem familiaridade com a publicação na Web. Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendencia em tomar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hj conseguem interpretar páginas web que estão longe de ser um código HTML válido.
